# SQL-based-data-analysis
# 📊 E-commerce Sales Analysis using SQL

## 📌 Overview

This project focuses on extracting meaningful business insights from sales data using SQL. It demonstrates how structured queries can be used to analyze large datasets and support data-driven decision-making.

---

## 🚀 Features

* SQL-based data extraction and analysis
* Aggregation and grouping for business metrics
* Time-based trend analysis
* Identification of top-performing categories, regions, and products

---

## 🛠️ Technologies Used

* SQLite (SQL queries)
* Python (Pandas for data loading)
* Jupyter Notebook

---

## 📊 Methodology

### 🔹 Data Preparation

* Cleaned dataset using Python
* Loaded data into SQLite database using `to_sql()`
* Verified data using SQL queries

---

### 🔹 SQL Analysis

#### ✅ Total Sales & Profit

* Used aggregate functions to compute overall performance

#### ✅ Category-wise Sales

* Grouped data to identify top-performing categories

#### ✅ Region-wise Profit

* Evaluated profitability across regions

#### ✅ Monthly Sales Trend

* Used date functions to analyze time-based patterns

#### ✅ Top 10 Products

* Identified high revenue-generating products

---

## 📈 Key Insights

* Technology and Furniture categories drive major revenue
* Some regions show lower profit margins despite strong sales
* Sales vary across months, indicating trend patterns
* Top products contribute significantly to overall revenue

---

## 📂 Dataset

* Global Superstore Dataset (50K+ records)

---

## ▶️ How to Run

1. Load dataset using Pandas
2. Create SQLite connection
3. Store data using `to_sql()`
4. Execute SQL queries for analysis

---

## 💡 Conclusion

This project highlights the use of SQL for efficient data querying, structured analysis, and business insight generation.
